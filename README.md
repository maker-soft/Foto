# Сайт «Выпускной альбом Новосибирск»

Готовая статическая сборка сайта и отдельной админки для GitHub Pages + Supabase.

## Начало работы

Подробная пошаговая инструкция находится в файлах:

- `ИНСТРУКЦИЯ_РАЗВЕРТЫВАНИЯ.md`;
- `Инструкция_развертывания_GitHub_Supabase.docx`.

Кратко необходимо:

1. создать проект и пользователя-администратора в Supabase;
2. заменить `admin@example.com` в `supabase/schema.sql` и выполнить SQL;
3. заполнить `assets/js/config.js`;
4. проверить сайт через локальный HTTP-сервер;
5. загрузить содержимое папки в корень GitHub-репозитория;
6. включить GitHub Pages для ветки `main`, папка `/(root)`.

Публичный сайт: `/`  
Админка: `/admin/`

Пароль не хранится в файлах проекта и проверяется через Supabase Auth. Не используйте Secret key или `service_role` в браузерном коде.
