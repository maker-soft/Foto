   - фотографий внутри альбомов;

3. При удалении альбома или подпапки фотографии, принадлежавшие им, также ставятся на безопасное удаление.

Как удалить фото
1. Откройте нужный раздел админки.
2. Нажмите «Удалить фото» или кнопку × на миниатюре.
3. Подтвердите удаление.
4. Обязательно нажмите «Сохранить на сайте».

Безопасность удаления
- Сначала сохраняется обновлённый контент сайта.
- После успешного сохранения файл удаляется из Supabase Storage.
- Если один файл используется в нескольких местах сайта, он не удаляется из Storage, пока остаётся хотя бы одна ссылка.
- Локальные изображения из папки assets нельзя физически удалить через Supabase, но их можно убрать со страниц сайта. Для физического удаления такого файла его нужно удалить из GitHub вручную.
