Версия 16

1. На смартфонах и планшетах кнопка «Меню» удалена.
2. Кнопки «Обо мне» и «Контакты» постоянно отображаются в верхней строке.
3. Если фотография удалена через админку, сайт больше не показывает заглушку и не сохраняет пустой блок изображения.
4. Макет автоматически перестраивается:
   - фото направления исчезает вместе с контейнером;
   - блок «Обо мне» и «Контакты» становится одноколоночным;
   - обложка альбома исчезает без пустого места;
   - альбом без обложки отображается как компактная текстовая карточка;
   - пустые папки фотогалереи не выводятся;
   - карточка направления без фото остаётся компактной кнопкой для перехода.

Для обновления GitHub заменить:
- index.html
- 404.html
- assets/js/site.js
- assets/css/site.css

Supabase и SQL изменять не требуется.
