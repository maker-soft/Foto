ПОЛНЫЙ ПЕРЕНОСИМЫЙ БЕКАП САЙТА С ФОТОГРАФИЯМИ — ВЕРСИЯ 12

Где находится:
Админка → Резервные копии.

КАК СОЗДАТЬ ПОЛНУЮ КОПИЮ
1. Сохраните текущие изменения сайта.
2. Откройте «Резервные копии».
3. Нажмите «Скачать полный ZIP с фото».
4. Не закрывайте страницу, пока скачиваются фотографии и формируется архив.
5. Сохраните ZIP-файл в надежном месте.

В ПОЛНЫЙ ZIP ВХОДЯТ
- весь контент и настройки дизайна;
- все фотографии, которые используются на страницах и в альбомах;
- таблица соответствия фотографий исходным адресам;
- HTML, CSS и JavaScript сайта и админки;
- SQL-файлы для создания и защиты базы Supabase;
- инструкция по переносу;
- переносимая версия контента со ссылками на файлы внутри архива.

КАК ПЕРЕНЕСТИ САЙТ НА ДРУГОЙ РЕСУРС
1. Разместите папку project из ZIP на новом GitHub Pages или другом статическом хостинге.
2. Создайте новый проект Supabase.
3. Выполните SQL-файлы из project/supabase.
4. Создайте администратора lena.foto@mail.ru.
5. Укажите адрес и публичный anon key нового Supabase в assets/js/config.js.
6. Откройте /admin/ нового сайта и войдите.
7. Откройте «Резервные копии» → «Восстановить полную ZIP-копию».
8. Выберите исходный ZIP-файл.
9. Админка загрузит фотографии в новый Supabase Storage, заменит все старые адреса и опубликует контент.

БЕЗОПАСНОСТЬ
В архив не включаются пароль администратора, service_role, секрет CAPTCHA и журнал попыток входа.
Файл config.js может содержать публичный anon key текущего проекта. При переносе его необходимо заменить данными нового проекта.

ОГРАНИЧЕНИЯ
- ZIP создается без сжатия фотографий, поэтому его размер примерно равен суммарному размеру изображений.
- Технический предел одного архива — 2 ГБ.
- Для больших фотогалерей рекомендуется создавать и восстанавливать копию на компьютере в Chrome, Edge или Safari.
- На iPhone/iPad большой архив может не сформироваться из-за ограничения оперативной памяти браузера.
- Статистика посещений и пользователи Supabase в переносимую копию не входят.

БЫСТРАЯ JSON-КОПИЯ
В админке сохранен прежний режим JSON. Он подходит для быстрого отката текстов и дизайна, но не содержит физических файлов фотографий.
